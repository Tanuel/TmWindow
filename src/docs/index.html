This library allows to create a simple window

Simple Usage


// create
let tmw = new TmWindow("My Window Title");

// set content
// You can either pass a string or a node
tmw.content = "My Window Content Lorem Ipsum";
tmw.content = document.createElement("p");

// show the window
tmw.open();
// close the window
tmw.close();
// minimize
tmw.minimize();
// remove the window element from the dom
tmw.remove();

Options

You can create the window with options instead of a title.

All options are optional


// create
let tmw = new TmWindow({
    title: "My Window",
    content: "Window Content",
    removeOnClose: false, // Don't remove the element from the DOM when closed
    style: {
        background: "green",
    },
    contain: true, // don't let the TmWindow move outside the Browserwindow
    resizable: true,
});

// Get and set options
tmw.setOption("removeOnClose", true)
   .setOption("resizable", false);
let option = tmw.getOption("removeOnClose");

// Change default options (will not update existing windows)
TmWindow.setDefaultOption("removeOnClose", true)
        .setDefaultOption("resizable", false;
let option = tmw.getDefaultOption("resizable");
